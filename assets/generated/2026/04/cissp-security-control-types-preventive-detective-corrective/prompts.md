# Image Prompts for CISSP #29: Security Control Types

## Hero Image Prompt

A layered diagram showing three concentric rings labeled Preventive, Detective, and Corrective Controls, with icons representing each (e.g., a lock for preventive, a magnifying glass for detective, a wrench for corrective). The background subtly hints at a business environment (office, cloud, or network elements). Professional, clean, and modern style.

## Inline Image 1 Prompt

A table or matrix visually mapping common security controls (e.g., firewall, IDS, backup, CCTV, access logs, incident response plan) to their control type(s): preventive, detective, corrective, deterrent, compensating. Use clear icons and color coding for each type.

## Inline Image 2 Prompt

A flowchart illustrating the lifecycle of a security incident: preventive control blocks most threats, detective control identifies a breach, corrective control restores normal operations. Use simple, clear arrows and icons for each stage.
